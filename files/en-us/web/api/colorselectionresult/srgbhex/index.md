{{APIRef("EyeDropper API")}}

The **`sRGBHex`** read-only property of the {{domxref("ColorSelectionResult")}} interface is a {{domxref("DOMString")}} of the color selected by the user with the eyedropper mode.

## Syntax

```js
result.sRGBHex;
```

### Value

A {{domxref("DOMString")}}.
